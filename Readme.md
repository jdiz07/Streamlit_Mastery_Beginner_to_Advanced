# ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Complete_Streamlit_With_Projects.git
```

Move into the project folder:

```bash
cd Complete_Streamlit_With_Projects
```

Install dependencies:

```bash
pip install streamlit
```

Run Streamlit app:

```bash
streamlit run app.py
```
